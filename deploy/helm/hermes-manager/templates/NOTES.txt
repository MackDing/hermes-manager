HermesManager has been installed!

=== Wait for the control plane (30-60s on first install) ===
{{- if .Values.postgres.enabled }}

  # Wait for PostgreSQL
  kubectl wait --for=condition=Ready pod \
    -l cnpg.io/cluster={{ include "hermesmanager.pgClusterName" . }} \
    -n {{ .Release.Namespace }} --timeout=5m
{{- end }}

  # Wait for hermesmanager
  kubectl wait --for=condition=Ready pod \
    -l app.kubernetes.io/name=hermesmanager \
    -n {{ .Release.Namespace }} --timeout=5m

=== Access the Web UI ===

  kubectl port-forward svc/{{ include "hermesmanager.fullname" . }} 8080:{{ .Values.service.port }} -n {{ .Release.Namespace }}

  Then open: http://localhost:8080

=== Admin Password ===

{{- if .Values.adminPassword }}

  You set an explicit admin password via Helm values.

{{- else }}

  An admin password was auto-generated. Retrieve it with:

  kubectl get secret {{ include "hermesmanager.fullname" . }} -n {{ .Release.Namespace }} \
    -o jsonpath='{.data.admin-password}' | base64 -d && echo

{{- end }}

=== Health Check ===

  curl http://localhost:8080/readyz

=== Quick Verification ===

  # After port-forwarding, verify the control plane is ready:
  curl -s http://localhost:8080/readyz | head -c 200

  # List loaded skills:
  curl -s http://localhost:8080/v1/skills | head -c 500

{{- if .Values.postgres.enabled }}

=== PostgreSQL ===

  Bundled PostgreSQL is managed by CloudNativePG.
  Cluster name: {{ include "hermesmanager.pgClusterName" . }}

  Check status:
  kubectl get cluster {{ include "hermesmanager.pgClusterName" . }} -n {{ .Release.Namespace }}

{{- end }}
